# ClawPage 技术栈

## 前端 (apps/web)
- **框架**: React 19
- **构建工具**: Vite
- **语言**: TypeScript
- **状态管理**: Zustand
- **UI 框架**: shadcn/ui
- **样式**: TailwindCSS
- **路由**: React Router v7
- **数据请求**: TanStack Query

## 后端 (workers/api)
- **运行时**: Cloudflare Workers
- **框架**: Hono
- **数据库**: Cloudflare D1 (SQLite)
- **文件存储**: Cloudflare R2

## 项目结构
- **包管理**: pnpm monorepo
- **共享包**: packages/shared (类型定义)
